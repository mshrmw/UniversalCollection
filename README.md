## Необходимо написать программу, реализующую универсальную коллекцию, которая будет представлять собой параметризованный класс, способный хранить объекты любого типа. 
Коллекция должна поддерживать добавление элементов, удаление элементов и получение элементов по индексу. 
Создайте параметризованный класс UniversalCollection<T>, который будет представлять собой универсальную коллекцию.
В классе UniversalCollection<T> реализуйте методы Add, Remove и Get для добавления элементов, удаления элементов и получения элемента по индексу соответственно.
В методе Main создайте объект класса UniversalCollection<T> для хранения целых чисел и добавьте несколько элементов в коллекцию.
Удалите элемент из коллекции и выведите оставшиеся элементы на экран.
Получите элемент по индексу и выведите его на экран.
При запуске программы должна быть создана универсальная коллекция для хранения целых чисел, в которую будут добавлены несколько элементов. Затем из коллекции должен быть удален элемент, и оставшиеся элементы должны быть выведены на экран. Наконец, элемент по индексу должен быть получен и выведен на экран.
Реализуйте пользовательский ввод в программу. Пользователь вводит число – количество элементов, которые пользователь хочет добавить в универсальную коллекцию, а затем вводит элементы. После добавления элементов необходимо предложить пользователю несколько действий: вывести все элементы, удалить определенный элемент, получить элемент по индексу.
